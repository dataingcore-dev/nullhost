غزو الفضائيين

بناء لعبة حاسوبية كاملة باستخدام مكتبة Pygame — الفصول 12 - 13 - 14

← العودة إلى الرئيسية

مرحباً بك في أول مشروع عملي في الكتاب! في هذا المشروع، سنبني لعبة كاملة تسمى "غزو الفضائيين" (Alien Invasion) باستخدام مكتبة Pygame. ستتعلم خلال ثلاثة فصول (12 و13 و14) كيفية بناء لعبة فيديو ثنائية الأبعاد خطوة بخطوة، بدءاً من إنشاء سفينة تطلق الرصاص، ومروراً بإضافة أسطول من الفضائيين، وانتهاءً بنظام تسجيل النقاط ومستويات الصعوبة.

هذا المشروع هو تتويج لمهاراتك في البرمجة كائنية التوجه (OOP) التي تعلمتها في الفصول السابقة. ستستخدم الكلاسات لتمثيل السفينة والرصاص والفضائيين، وستتعامل مع الأحداث (events) وحلقات اللعبة الرئيسية (game loop)، وستتعلم كيفية إدارة الشاشة والرسوم المتحركة.

الفصل 12: سفينة تطلق الرصاص

12.1 تخطيط المشروع

قبل كتابة أي كود، من المهم التخطيط للمشروع. لعبة "غزو الفضائيين" ستكون لعبة ثنائية الأبعاد حيث يتحكم اللاعب بسفينة في أسفل الشاشة، وتظهر الفضائيون من الأعلى ويتحركون ببطء نحو الأسفل. مهمة اللاعب هي إطلاق الرصاص على الفضائيين وتجنب وصولهم إلى سفينته.

سنقسم الكود إلى عدة ملفات لتنظيم العمل:

  • alien_invasion.py — الملف الرئيسي للعبة
  • settings.py — كلاس الإعدادات
  • ship.py — كلاس السفينة
  • bullet.py — كلاس الرصاصة
  • alien.py — كلاس الفضائي
  • game_stats.py — إحصائيات اللعبة
  • scoreboard.py — لوحة النتائج
  • button.py — زر اللعب

12.2 تثبيت Pygame

Pygame هي مكتبة مفتوحة المصدر تتيح بناء ألعاب فيديو بلغة بايثون. توفر أدوات للتعامل مع الرسومات والصوت وأحداث لوحة المفاتيح والفأرة.

# تثبيت Pygame عبر pip python3 -m pip install pygame

لتأكيد التثبيت، يمكنك تشغيل:

python3 -m pygame --version

12.3 إنشاء نافذة اللعبة وحلقة اللعبة الرئيسية

كل لعبة Pygame تبدأ بإنشاء نافذة وحلقة رئيسية (game loop) تدير الأحداث وتحدث الرسوم. إليك الهيكل الأساسي:

import pygame import sys class AlienInvasion: def __init__(self): pygame.init() self.screen = pygame.display.set_mode((1200, 800)) pygame.display.set_caption("Alien Invasion") def run_game(self): while True: for event in pygame.event.get(): if event.type == pygame.QUIT: sys.exit() pygame.display.flip() if __name__ == '__main__': ai = AlienInvasion() ai.run_game()

pygame.init() تهيئ جميع وحدات Pygame. pygame.display.set_mode() تنشئ نافذة بأبعاد 1200×800 بكسل. الحلقة الرئيسية تستمر في العمل حتى يغلق المستخدم النافذة، وتستمع إلى الأحداث (مثل ضغطات المفاتيح) وتحدث الشاشة.

12.4 كلاس الإعدادات (Settings)

بدلاً من توزيع الثوابت في كل أرجاء الكود، نضعها في كلاس إعدادات مركزي. هذا يجعل تعديل اللعبة سهلاً للغاية:

class Settings: def __init__(self): self.screen_width = 1200 self.screen_height = 800 self.bg_color = (230, 230, 230) self.ship_speed = 1.5 self.bullet_speed = 2.5 self.bullet_width = 3 self.bullet_height = 15 self.bullet_color = (60, 60, 60) self.bullets_allowed = 3 self.alien_speed = 1.0 self.fleet_drop_speed = 10 self.fleet_direction = 1 # 1: يمين, -1: يسار

12.5 إضافة صورة السفينة

نحتاج إلى صورة للسفينة. يمكنك استخدام أي صورة صغيرة (مثل 60×80 بكسل). ضعها في مجلد images/ داخل مجلد المشروع. إليك كلاس السفينة:

import pygame class Ship: def __init__(self, ai_game): self.screen = ai_game.screen self.settings = ai_game.settings self.screen_rect = ai_game.screen.get_rect() self.image = pygame.image.load('images/ship.bmp') self.rect = self.image.get_rect() self.rect.midbottom = self.screen_rect.midbottom self.x = float(self.rect.x) self.moving_right = False self.moving_left = False def blitme(self): self.screen.blit(self.image, self.rect)

12.6 إعادة التكوين (Refactoring)

مع نمو اللعبة، يصبح الكود في الملف الرئيسي كبيراً جداً. لذا نُعيد تنظيم الكود بنقل أجزاء منه إلى طرق (methods) أصغر في كلاسات منفصلة. مثلاً ننقل معالجة الأحداث إلى _check_events() وتحديث الشاشة إلى _update_screen():

def _check_events(self): for event in pygame.event.get(): if event.type == pygame.QUIT: sys.exit() elif event.type == pygame.KEYDOWN: if event.key == pygame.K_RIGHT: self.ship.moving_right = True elif event.key == pygame.K_LEFT: self.ship.moving_left = True elif event.type == pygame.KEYUP: if event.key == pygame.K_RIGHT: self.ship.moving_right = False elif event.key == pygame.K_LEFT: self.ship.moving_left = False

12.7 إطلاق الرصاص

الرصاصات عبارة عن مستطيلات (rectangles) تتحرك للأعلى. نستخدم مجموعة (group) لتخزين الرصاصات النشطة:

class Bullet(pygame.sprite.Sprite): def __init__(self, ai_game): super().__init__() self.screen = ai_game.screen self.settings = ai_game.settings self.color = self.settings.bullet_color self.rect = pygame.Rect(0, 0, self.settings.bullet_width, self.settings.bullet_height) self.rect.midtop = ai_game.ship.rect.midtop self.y = float(self.rect.y) def update(self): self.y -= self.settings.bullet_speed self.rect.y = self.y def draw_bullet(self): pygame.draw.rect(self.screen, self.color, self.rect)
ملاحظة: استخدام pygame.sprite.Sprite يتيح لنا تجميع الرصاصات في مجموعة pygame.sprite.Group، مما يسهل تحديث ورسم جميع الرصاصات دفعة واحدة.

الفصل 13: الفضائيون

13.1 إنشاء كلاس الفضائي

الفضائي يشبه السفينة في بنيته — لديه صورة ومستطيل (rect). لكنه يتحرك بطريقة مختلفة:

class Alien(pygame.sprite.Sprite): def __init__(self, ai_game): super().__init__() self.screen = ai_game.screen self.settings = ai_game.settings self.image = pygame.image.load('images/alien.bmp') self.rect = self.image.get_rect() self.rect.x = self.rect.width self.rect.y = self.rect.height self.x = float(self.rect.x) def check_edges(self): screen_rect = self.screen.get_rect() return (self.rect.right >= screen_rect.right or self.rect.left <= 0) def update(self): self.x += self.settings.alien_speed * self.settings.fleet_direction self.rect.x = self.x

13.2 إنشاء أسطول الفضائيين

نحتاج إلى إنشاء صفوف من الفضائيين تملأ عرض الشاشة. نحسب عدد الفضائيين الذين يمكن وضعهم في صف بناءً على عرض الشاشة وعرض الفضائي:

def _create_fleet(self): alien = Alien(self) alien_width, alien_height = alien.rect.size available_space_x = self.settings.screen_width - (2 * alien_width) number_aliens_x = available_space_x // (2 * alien_width) ship_height = self.ship.rect.height available_space_y = (self.settings.screen_height - (3 * alien_height) - ship_height) number_rows = available_space_y // (2 * alien_height) for row_number in range(number_rows): for alien_number in range(number_aliens_x): self._create_alien(alien_number, row_number)

13.3 تحريك الأسطول

يتحرك الأسطول بأكمله يميناً حتى يصل إلى حافة الشاشة، ثم ينزل للأسفل ويتجه لليسار. هذا يُعطي إحساساً بحركة منظمة تشبه لعبة Space Invaders الكلاسيكية.

المفتاح هو متغير fleet_direction: عندما يكون 1 يتحرك الأسطول لليمين، وعندما يكون -1 يتحرك لليسار. عندما يلمس أي فضائي حافة الشاشة، يتغير الاتجاه وينزل الأسطول للأسفل بمقدار fleet_drop_speed.

13.4 إطلاق النار على الفضائيين والتصادمات

عندما تطلق رصاصة وتصيب فضائياً، نحتاج إلى إزالة كليهما. يستخدم Pygame دوال التصادم المدمجة:

def _check_bullet_alien_collisions(self): collisions = pygame.sprite.groupcollide( self.bullets, self.aliens, True, True) if not self.aliens: self.bullets.empty() self._create_fleet()

groupcollide() تفحص التصادم بين مجموعتين. المعامل الثالث (True) يعني احذف الرصاصة عند التصادم، والرابع يعني احذف الفضائي. إذا تم تدمير كل الفضائيين، نُفرغ الرصاصات وننشئ أسطولاً جديداً (مستوى جديد).

13.5 نهاية اللعبة

تنتهي اللعبة عندما يصل فضائي إلى أسفل الشاشة أو عندما يصطدم فضائي بالسفينة. في هذه الحالة، نقلل عدد السفن المتاحة. إذا وصل العدد إلى صفر، تنتهي اللعبة:

def _check_aliens_bottom(self): screen_rect = self.screen.get_rect() for alien in self.aliens.sprites(): if alien.rect.bottom >= screen_rect.bottom: self._ship_hit() break

الفصل 14: تسجيل النقاط

14.1 زر اللعب (Play Button)

نحتاج إلى شاشة بداية بسيطة تظهر للمستخدم قبل بدء اللعبة. ننشئ كلاس Button يرسم مستطيلاً مع نص:

class Button: def __init__(self, ai_game, msg): self.screen = ai_game.screen self.screen_rect = self.screen.get_rect() self.width, self.height = 200, 50 self.button_color = (0, 255, 0) self.text_color = (255, 255, 255) self.font = pygame.font.SysFont(None, 48) self.rect = pygame.Rect(0, 0, self.width, self.height) self.rect.center = self.screen_rect.center self._prep_msg(msg) def _prep_msg(self, msg): self.msg_image = self.font.render(msg, True, self.text_color, self.button_color) self.msg_image_rect = self.msg_image.get_rect() self.msg_image_rect.center = self.rect.center def draw_button(self): self.screen.fill(self.button_color, self.rect) self.screen.blit(self.msg_image, self.msg_image_rect)

14.2 مستويات الصعوبة المتزايدة

مع كل مستوى جديد، نزيد سرعة اللعبة تدريجياً لجعلها أكثر تحدياً. نستخدم طريقة increase_speed() في كلاس الإعدادات:

def increase_speed(self): self.ship_speed *= self.speedup_scale self.bullet_speed *= self.speedup_scale self.alien_speed *= self.speedup_scale self.alien_points = int(self.alien_points * self.score_scale)

14.3 لوحة النتائج (Scoreboard)

نعرض النقاط الحالية وأعلى نتيجة والمستوى الحالي وعدد السفن المتبقية على الشاشة. نستخدم pygame.font لعرض النصوص:

class Scoreboard: def __init__(self, ai_game): self.screen = ai_game.screen self.screen_rect = self.screen.get_rect() self.settings = ai_game.settings self.stats = ai_game.stats self.text_color = (30, 30, 30) self.font = pygame.font.SysFont(None, 48) self.prep_score() self.prep_high_score() self.prep_level() self.prep_ships() def prep_score(self): score_str = str(self.stats.score) self.score_image = self.font.render(score_str, True, self.text_color, self.settings.bg_color) self.score_rect = self.score_image.get_rect() self.score_rect.right = self.screen_rect.right - 20 self.score_rect.top = 20 def show_score(self): self.screen.blit(self.score_image, self.score_rect) self.screen.blit(self.high_score_image, self.high_score_rect) self.screen.blit(self.level_image, self.level_rect) self.ships.draw(self.screen)

14.4 حفظ أعلى نتيجة

نستخدم ملف JSON لحفظ أعلى نتيجة حتى تبقى محفوظة عند إغلاق اللعبة وإعادة فتحها:

import json from pathlib import Path def get_stored_high_score(): path = Path('high_score.json') if path.exists(): contents = path.read_text() return json.loads(contents) return 0

المفاهيم الرئيسية

📝 المفاهيم التي تعلمتها في هذا المشروع

  • Pygame: مكتبة لبناء ألعاب فيديو ثنائية الأبعاد بلغة بايثون
  • حلقة اللعبة الرئيسية (Game Loop): الحلقة التي تدير أحداث اللعبة وتحديثاتها باستمرار
  • الأحداث (Events): تفاعلات المستخدم مثل الضغط على المفاتيح أو النقر بالماوس
  • كلاس Sprite: كلاس أساسي في Pygame لتمثيل الكائنات المرئية
  • المجموعات (Groups): لتجميع وإدارة الكائنات المتشابهة (مثل الرصاص والفضائيين)
  • اكتشاف التصادم (Collision Detection): اكتشاف متى يتصادم كائنان باستخدام groupcollide() و spritecollideany()
  • مستويات الصعوبة: زيادة سرعة اللعبة تدريجياً مع كل مستوى جديد
  • حفظ البيانات: استخدام JSON لحفظ أعلى نتيجة بين جلسات اللعب
نصيحة ختامية: لا تتوقف عند الكود الموجود في الكتاب. جرب إضافة ميزاتك الخاصة! أضف أصواتاً للمؤثرات، غيّر ألوان الخلفية، أو أنشئ أنواعاً جديدة من الفضائيين. أفضل طريقة للتعلم هي تعديل الكود وتجربة أفكار جديدة.
