تصور البيانات

إنشاء رسوم بيانية تفاعلية، تحميل البيانات من الإنترنت، والعمل مع واجهات برمجة التطبيقات — الفصول 15 - 16 - 17

← العودة إلى الرئيسية

في عالم اليوم، البيانات في كل مكان. القدرة على فهمها وتصورها هي مهارة أساسية لأي مبرمج. في هذا المشروع، ستتعلم كيفية إنشاء رسوم بيانية جميلة باستخدام مكتبات مثل Matplotlib و Plotly (Pygal في النسخة الأصلية)، وكيفية تحميل البيانات من ملفات CSV و JSON، وكيفية جلب البيانات الحية من واجهات برمجة التطبيقات (APIs) مثل GitHub API.

هذا المشروع سيفتح لك عالماً جديداً من الإمكانيات — من تحليل بيانات الطقس إلى متابعة أخبار التقنية، كل ذلك باستخدام بايثون.

الفصل 15: توليد البيانات

15.1 تثبيت Matplotlib و Plotly

قبل أن نبدأ في إنشاء الرسوم البيانية، نحتاج إلى تثبيت المكتبات المناسبة:

# تثبيت Matplotlib python3 -m pip install --upgrade matplotlib # تثبيت Plotly python3 -m pip install --upgrade plotly # تثبيت Pygal (إذا أردت استخدامه بديلاً) python3 -m pip install --upgrade pygal

15.2 رسم مخطط خطي بسيط باستخدام pyplot

matplotlib.pyplot هي وحدة رسوم بيانية شهيرة. لعمل أبسط رسم، نحتاج إلى قائمة من الأرقام للمحورين X و Y:

import matplotlib.pyplot as plt squares = [1, 4, 9, 16, 25] input_values = [1, 2, 3, 4, 5] plt.style.use('seaborn') fig, ax = plt.subplots() ax.plot(input_values, squares, linewidth=3) # تسمية المحاور والعنوان ax.set_title("Square Numbers", fontsize=24) ax.set_xlabel("Value", fontsize=14) ax.set_ylabel("Square of Value", fontsize=14) ax.tick_params(axis='both', labelsize=14) plt.show()

plt.style.use('seaborn') يطبق نمطاً جميلاً على الرسم. fig, ax = plt.subplots() ينشئ الشكل (figure) والمحاور (axes). بعد ذلك، نضيف البيانات ونخصص المخطط قبل عرضه بـ plt.show().

15.3 المخططات المبعثرة (Scatter Plots)

المخططات المبعثرة مفيدة لرؤية العلاقات بين مجموعتين من البيانات. نستخدم ax.scatter():

import matplotlib.pyplot as plt x_values = range(1, 1001) y_values = [x**2 for x in x_values] plt.style.use('seaborn') fig, ax = plt.subplots() ax.scatter(x_values, y_values, c=y_values, cmap=plt.cm.Blues, s=10) ax.set_title("Square Numbers", fontsize=24) ax.set_xlabel("Value", fontsize=14) ax.set_ylabel("Square of Value", fontsize=14) ax.axis([0, 1100, 0, 1100000]) plt.show()

لاحظ كيف استخدمنا c=y_values, cmap=plt.cm.Blues لتلوين النقاط حسب قيمها — هذا يضيف بُعداً جمالياً ومعلوماتياً إضافياً للمخطط.

15.4 المشي العشوائي (Random Walk)

المشي العشوائي يحاكي مساراً عشوائياً — مثلاً حركة جزيء في سائل أو تقلبات سوق الأسهم. ننشئ كلاس RandomWalk يولد نقاطاً عشوائية:

from random import choice class RandomWalk: def __init__(self, num_points=5000): self.num_points = num_points self.x_values = [0] self.y_values = [0] def fill_walk(self): while len(self.x_values) < self.num_points: x_direction = choice([1, -1]) x_distance = choice([0, 1, 2, 3, 4]) x_step = x_direction * x_distance y_direction = choice([1, -1]) y_distance = choice([0, 1, 2, 3, 4]) y_step = y_direction * y_distance if x_step == 0 and y_step == 0: continue x = self.x_values[-1] + x_step y = self.y_values[-1] + y_step self.x_values.append(x) self.y_values.append(y)

ثم نرسم المسار الناتج باستخدام ax.scatter() مع تدرج لوني يظهر تقدم المسار:

rw = RandomWalk() rw.fill_walk() plt.style.use('classic') fig, ax = plt.subplots() point_numbers = range(rw.num_points) ax.scatter(rw.x_values, rw.y_values, c=point_numbers, cmap=plt.cm.Blues, edgecolors='none', s=15) ax.scatter(0, 0, c='green', edgecolors='none', s=100) ax.scatter(rw.x_values[-1], rw.y_values[-1], c='red', edgecolors='none', s=100) ax.get_xaxis().set_visible(False) ax.get_yaxis().set_visible(False) plt.show()

15.5 استخدام Plotly للرسوم التفاعلية

بينما Matplotlib ممتاز للرسوم الثابتة، Plotly يتيح رسوماً تفاعلية يمكن للمستخدم zoom و hover عليها. هنا مثال باستخدام مكعبات الأعداد:

from plotly.graph_objs import Bar, Layout from plotly import offline x_values = list(range(1, 11)) y_values = [x**3 for x in x_values] data = [Bar(x=x_values, y=y_values)] x_axis_config = {'title': 'Number'} y_axis_config = {'title': 'Cube'} my_layout = Layout(title='Cubes', xaxis=x_axis_config, yaxis=y_axis_config) offline.plot({'data': data, 'layout': my_layout}, filename='cubes.html')

الرسوم البيانية من Plotly تُحفظ كملفات HTML تفاعلية — يمكن فتحها في أي متصفح وتسمح بالتفاعل مع البيانات.

ملاحظة: في النسخة الأصلية من الكتاب، استخدم المؤلف مكتبة Pygal بدلاً من Plotly. كلتا المكتبتين ممتازتان، لكن Plotly أحدث وأكثر شيوعاً حالياً. اختر ما يناسبك.

الفصل 16: تحميل البيانات

16.1 تنسيق CSV

ملفات CSV (Comma-Separated Values) هي تنسيق شائع لتخزين البيانات الجدولية. كل سطر يمثل صفاً، والقيم مفصولة بفواصل. مثلاً ملف بيانات الطقس:

# مثال من ملف weather_data.csv STATION,NAME,DATE,PRCP,TAVG,TMAX,TMIN USC00458830,San Francisco,2023-01-01,0.01,12.8,15.6,10.0 USC00458830,San Francisco,2023-01-02,0.00,13.3,16.7,9.4

16.2 قراءة ملفات CSV باستخدام csv module

نستخدم وحدة csv المدمجة في بايثون لقراءة هذه الملفات:

import csv from pathlib import Path path = Path('weather_data/san_francisco_2023.csv') lines = path.read_text().splitlines() reader = csv.reader(lines) header_row = next(reader) # طباعة أسماء الأعمدة مع فهارسها for index, column_header in enumerate(header_row): print(index, column_header) # استخراج درجات الحرارة العظمى highs = [] for row in reader: high = int(row[6]) # عمود TMAX highs.append(high)

16.3 رسم بيانات الطقس

بعد استخراج البيانات، نرسمها باستخدام Matplotlib:

plt.style.use('seaborn') fig, ax = plt.subplots() ax.plot(highs, color='red', linewidth=1) # تنسيق المحور x لعرض التواريخ ax.set_title("Daily High Temperatures, San Francisco 2023", fontsize=24) ax.set_xlabel('', fontsize=16) ax.set_ylabel("Temperature (F)", fontsize=16) ax.tick_params(axis='both', which='major', labelsize=16) plt.show()

16.4 تنسيق JSON

JSON (JavaScript Object Notation) هو تنسيق شائع آخر لتبادل البيانات، خاصة في تطبيقات الويب. مثلاً بيانات إعمار سكان المدن:

[ { "Country": "World", "Population": 7890000000, "Cities": [ { "City": "Tokyo", "Population": 37393128 }, { "City": "Delhi", "Population": 31181376 } ] } ]

16.5 معالجة بيانات JSON

نستخدم وحدة json المدمجة:

import json from pathlib import Path path = Path('population_data.json') contents = path.read_text() population_data = json.loads(contents) # طباعة أسماء المدن for country_data in population_data: print(country_data['Country']) for city_data in country_data['Cities']: print(f" {city_data['City']}: {city_data['Population']}")

الفصل 17: العمل مع APIs

17.1 ما هي واجهة برمجة التطبيقات (API)؟

API (Application Programming Interface) هي واجهة تسمح لبرنامجين بالتواصل مع بعضهما البعض. في سياق الويب، تسمح APIs للمطورين بجلب البيانات من مواقع أخرى. مثلاً GitHub API يتيح لك جلب معلومات عن المستودعات والمستخدمين والمساهمات.

17.2 إجراء طلب API باستخدام requests

نستخدم مكتبة requests لإجراء طلبات HTTP:

import requests url = 'https://api.github.com/search/repositories?q=language:python&sort=stars' headers = {'Accept': 'application/vnd.github.v3+json'} r = requests.get(url, headers=headers) print(f"Status code: {r.status_code}") # تحويل الرد إلى JSON response_dict = r.json() print(f"Total repositories: {response_dict['total_count']}") # استعراض تفاصيل المستودعات repo_dicts = response_dict['items'] print(f"Repositories returned: {len(repo_dicts)}") for repo_dict in repo_dicts[:5]: print("\nName:", repo_dict['name']) print("Owner:", repo_dict['owner']['login']) print("Stars:", repo_dict['stargazers_count']) print("Description:", repo_dict['description'])

نستخدم requests.get() لإرسال طلب GET إلى URL المحدد. الخيار headers يخبر GitHub API بأننا نريد الرد بتنسيق JSON. الكود r.status_code يخبرنا ما إذا كان الطلب ناجحاً (200 يعني نجاح).

ملاحظة: تأكد من تثبيت مكتبة requests باستخدام python3 -m pip install requests قبل استخدامها.

17.3 معالجة نتائج API

بعد جلب البيانات، نستخرج المعلومات التي تهمنا. في مثالنا، نأخذ أسماء المستودعات وعدد النجوم والوصف. لكن يمكننا استخراج أي حقل متاح في JSON.

إليك كيفية استخراج الروابط (URLs) أيضاً:

for repo_dict in repo_dicts[:5]: print("URL:", repo_dict['html_url']) print("Created:", repo_dict['created_at']) print("Updated:", repo_dict['updated_at']) print("Language:", repo_dict['language'])

17.4 تصور نتائج API

الآن نأخذ معلومات النجوم ونعرضها في مخطط شريطي باستخدام Plotly:

from plotly.graph_objs import Bar, Layout from plotly import offline repo_names, stars = [], [] for repo_dict in repo_dicts[:30]: repo_names.append(repo_dict['name']) stars.append(repo_dict['stargazers_count']) data = [Bar(x=repo_names, y=stars)] x_axis_config = {'title': 'Repository', 'tickangle': -45} y_axis_config = {'title': 'Stars'} my_layout = Layout(title='Most-Starred Python Repos on GitHub', xaxis=x_axis_config, yaxis=y_axis_config) offline.plot({'data': data, 'layout': my_layout}, filename='python_repos.html')

17.5 واجهات API أخرى

هناك آلاف APIs المجانية التي يمكنك استخدامها:

  • GitHub API — معلومات عن المستودعات والمستخدمين
  • News API — أخبار من مصادر مختلفة
  • OpenWeatherMap — بيانات الطقس الحالية
  • Twitter API — التغريدات والاتجاهات
  • NYT API — مقالات نيويورك تايمز

المفاهيم الرئيسية

📝 المفاهيم التي تعلمتها في هذا المشروع

  • Matplotlib: مكتبة رسوم بيانية ثابتة قوية ومرنة
  • Plotly / Pygal: مكتبات رسوم بيانية تفاعلية قابلة للتصدير إلى HTML
  • المخطط المبعثر (Scatter Plot): عرض العلاقات بين متغيرين باستخدام نقاط
  • المشي العشوائي (Random Walk): محاكاة مسارات عشوائية لتقليد الظواهر الطبيعية
  • تنسيق CSV: تنسيق نصي للبيانات الجدولية — قراءته وتحليله مع بايثون
  • تنسيق JSON: تنسيق لتبادل البيانات — معالجته وتحويله إلى قواميس بايثون
  • API (واجهة برمجة التطبيقات): جلب بيانات حية من الإنترنت باستخدام طلبات HTTP
  • مكتبة requests: إجراء طلبات HTTP بسهولة في بايثون
نصيحة ختامية: تصور البيانات هو مهارة مطلوبة بشدة في سوق العمل. جرب باستمرار تطبيق ما تعلمته على بيانات حقيقية تجدها على الإنترنت. كلما تدربت أكثر، كلما أصبحت الرسوم البيانية التي تنشئها أكثر إتقاناً وإفادة.
