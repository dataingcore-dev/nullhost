التمرين 5: دالة حساب الضريبة
اكتب دالة calc_tax(price, rate=0.15) تعيد السعر بعد الضريبة.
def calc_tax(price, rate=0.15): return price * (1 + rate) print(calc_tax(100)) # 115.0 print(calc_tax(100, 0.1)) # 110.0

🔷 OOP — البرمجة كائنية التوجه

التمرين 7: كلاس BankAccount
أنشئ كلاس BankAccount بالدوال التالية:
deposit(amount) — إيداع
withdraw(amount) — سحب (إذا كان الرصيد كافياً)
get_balance() — عرض الرصيد
class BankAccount: def __init__(self, owner, balance=0): self.owner = owner self.balance = balance def deposit(self, amount): self.balance += amount print(f"تم إيداع {amount}. الرصيد: {self.balance}") def withdraw(self, amount): if amount > self.balance: print("رصيد غير كافٍ!") else: self.balance -= amount print(f"تم سحب {amount}. الرصيد: {self.balance}") def get_balance(self): return self.balance acc = BankAccount("Ahmed", 1000) acc.deposit(500) acc.withdraw(200) print(acc.get_balance()) # 1300
التمرين 8: وراثة — SavingAccount
أنشئ كلاس SavingAccount يرث من BankAccount ويضيف خاصية interest_rate ودالة add_interest().
class SavingAccount(BankAccount): def __init__(self, owner, balance=0, interest_rate=0.05): super().__init__(owner, balance) self.interest_rate = interest_rate def add_interest(self): interest = self.balance * self.interest_rate self.deposit(interest) print(f"فائدة {interest} أضيفت.") save = SavingAccount("Sara", 2000, 0.05) save.add_interest() print(save.get_balance()) # 2100.0

🗄️ SQL — قواعد البيانات

التمرين 9: استعلام JOIN
لديك جدولان: customers(id, name) و orders(id, customer_id, total). اكتب استعلاماً يعيد أسماء الزبائن مع مجموع مشترياتهم.
SELECT c.name, SUM(o.total) as total_spent FROM customers c JOIN orders o ON c.id = o.customer_id GROUP BY c.id, c.name ORDER BY total_spent DESC;

🚀 FastAPI

التمرين 10: API for To-Do List
أنشئ API بسيط للمهام بمسارين:
GET /todos — يعرض جميع المهام
POST /todos — يضيف مهمة جديدة (body: {title, completed})
from fastapi import FastAPI from pydantic import BaseModel app = FastAPI() class Todo(BaseModel): title: str completed: bool = False todos = [] @app.get("/todos") def list_todos(): return todos @app.post("/todos") def create_todo(todo: Todo): todos.append(todo) return todo

⚡ Redis

التمرين 11: عداد زيارات الصفحة
استخدم Redis لحساب عدد زيارات كل صفحة. زوّد العداد في كل طلب وأعده.
import redis from fastapi import FastAPI app = FastAPI() r = redis.Redis(host='localhost', port=6379, db=0) @app.get("/page/{page_id}") def visit_page(page_id: str): count = r.incr(f"page_views:{page_id}") return {"page": page_id, "views": count}

🤖 أساسيات AI

التمرين 13: تشابه النصوص
استخدم sentence-transformers لحساب التشابه بين جملتين.
from sentence_transformers import SentenceTransformer from sklearn.metrics.pairwise import cosine_similarity model = SentenceTransformer('all-MiniLM-L6-v2') sentences = ["أحب تعلم البرمجة", "البرمجة شغفي"] emb = model.encode(sentences) sim = cosine_similarity([emb[0]], [emb[1]])[0][0] print(f"نسبة التشابه: {sim:.2f}")
